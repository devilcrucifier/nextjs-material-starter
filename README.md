# nextjs-redux-material-starter

Starter for Nextjs, Redux and Material UI

## Development

Install project dependencies:

```javascript
npm install
```

To run the application in development mode

```javascript
npm run dev
```

To run the application in production enviorment

```javascript
npm run build
npm run start
```
